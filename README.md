## What's in this repo

| Path | Description |
| --- | --- |
| `Joystick Gremlin Profile [ENH][GF]…[R14].xml` | The R14 JG profile — physical buttons → vJoy |
| `layout_ENH_GF_*_LIVE_exported.xml` | Star Citizen control layout — vJoy → game actions |
| `layout_SUB_Clear_Bindings_*_exported.xml` | Clean-slate layout for baseline testing |
| `Tools/` | Bindings Toolkit — MFD fix, axis invert, diagnostics |
| `Audio Files/` | Optional mode-audio prompts |

## Setup

Load the profile in **Joystick Gremlin R14**, inject the layout into Star Citizen, and fly. The full walkthrough is in the [setup guide](https://subliminalstv.github.io/Subs-Curated-Bindings/general-setup/download/).

> ⚠️ This is the **bleeding-edge / contributor** copy — files here may be experimental or mid-patch. For fully tested, plug-and-play binds, use **[subliminal.gg/bindings](https://subliminal.gg/bindings/)**.
